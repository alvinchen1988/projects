For those who are not sure about the answers, please allow me to give a breif introduction of Javascript and jQuery.

What is Javascript?

Javascript is a client-side language that controls the behaviors of webpages. By manipulating the Document Object Model(DOM), you can make webpages that interact with the users. As web applications become more and more popular, javascript has become an essential skill in a web developer's toolkit.

What is jQuery?

jQuery is a lightweight, "write less, do more", JavaScript library. It makes things like HTML document traversal and manipulation, event handling, animation, and Ajax much simpler with an easy-to-use API that works across a multitude of browsers.

Why jQuery?

The purpose of jQuery is to make it much easier to use JavaScript on your website. jQuery takes a lot of common tasks that require many lines of JavaScript code to accomplish, and wraps them into methods that you can call with a single line of code. It supports all main-stream browsers and has plugins for almost any task out there.

Lesson 1: The Image Inspector

In our first lesson, I will show you step by step how to use a plugin called Pep. Pep is a lightweight jQuery plugin which turns any DOM element into a draggable object. It works across mostly all browsers, from old to new, from touch to click. Follow the steps below to see how i turn a regular image into an image with an inspector.

Demo 1

Step 1 - Download the plugin from here and put the jquery.pep.js file in your javascript folder. Don't forget to include the jquery.min.js file and your own js script as well. Link them in your html markup before the closing body tag.

	<script src="scripts/jquery.pep.js"></script>
	<script src="scripts/script.js"></script>

	var $result    = $('.image.result');
	var $viewer    = $('.image.viewer');
	var $magnifier = $viewer.find('.magnifier');

  	constrainTo: 'parent',

  	var y = ( (pos.top + $magnifier.outerHeight()/2 )/$viewer.height() )*100 + "%";

 	 $result.css('background-position', [x, y].join(' '));

 	 }
	})

Lesson 2: Use Alertify To Make Your Browser Dialogs Fancier

In our second lesson, I will show you step by step how to use a plugin called Alertify. Alertify.js is a lightweight jQuery plugin which makes the default browser dialogs much prettier. It works across mostly all browsers, from old to new, from touch to click. Try out the demo below and follow step by step to see how it works.

Demo 2

Step 1 - Download the plugin from here and put the alertify.min.js file in your javascript folder. Also put the alertify.core.css and alertify.default.css into your css folder. Don't forget to include the jquery.min.js file and your own js script as well. Your markup should look like this:

				
	<head>
	<link rel="stylesheet" href="styles/alertify.core.css">
	<link rel="stylesheet" href="styles/alertify.default.css">
	</head>

	<script src="http://ajax.googleapis.com/ajax/libs/jquery/1.10.2/jquery.min.js"></script>
    <script src="scripts/alertify.min.js"></script>
	<script src="scripts/script.js"></script>

Step 2 - In you body markup, you can write the html code to your needs. For demonstration purpose i made three buttons. So this is what my html looks like:

				

	<div class="demo2">
  	 <button type="button" id="prompt">Hello</button>
  	 <button type="button" id="confirm">Bonjour</button>
  	 <button type="button" id="forever">你好</button>
	</div>

Step 3 - Style your CSS(Optional) - The default CSS styles are actually quite good. I only changed the margin-top property of the nav element. But if you want to tweak it a bit more. Feel free to do so in the alertify.default.css file.

				
	nav.alertify-buttons{
	margin-top: 18px;
	}

Step 4 - Write javascript in the javascript file you created

				
	function reset () {
	$("#toggleCSS").attr("href", "../styles/alertify.default.css");
	alertify.set({
	labels : {
	ok     : "OK",
	cancel : "Cancel"
	},
	delay : 5000,
	buttonReverse : false,
	// buttonFocus   : "ok"
	});
	}

	$("#confirm").on( 'click', function () {
	reset();
	alertify.set({ labels: { ok: "Yes, indeed!", cancel: "Piece of cake.." } });
	alertify.confirm("Do you find coding javascript challenging?", function (e) {
	if (e) {
	alertify.success("Don't be discouraged! Follow along and you will be just fine.");
	} else {
	alertify.error("I'm impressed by your coding skill.");
	}
	});
	return false;
	});

	$("#prompt").on( 'click', function () {
	reset();
	alertify.prompt("Hello! Thank you for dropping by my JS tutorial page. May i have your name please?", function (e, str) {
	if (e) {
	alertify.success("How are you doing today? " + str);
	} else {
	alertify.error("I am sorry to see you go.");
	}
	}, "enter your name");
	return false;
	});

	$("#forever").on( 'click', function () {
	reset();
	alertify.set({ labels: { ok: "It is helpful", cancel: "Not so much" }, delay: 10000, });
	alertify.confirm("Is this JS toturial helpful to you?", function (e) {
	if (e) {
	alertify.success("I'm glad to hear. Please share this website with your friends so more people can learn coding.");
	} else {
	alertify.error("I'm sorry it did't help you. Please help me improve this website by filling out the form below.");
	}
	});
	return false;
	});

Lesson 3: The draggable Widgets

In this last lesson, I will show you step by step how to use a plugin called Gridster.js. Gridster is a lightweight jQuery plugin which can turn any DOM element into widgets that are draggable. It is mostly used to make expandable and movable divs. It works across mostly all browsers, from old to new, from touch to click.
